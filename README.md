# MarkDown  
## 定义  
轻量级标记语言  
## 优点  
- 纯文本内容，兼容所有的文本编辑器与文字出路软件  
- 可以放到版本管理系统中，最总历史变更  
- 轻松的导出HTML，PDF和本身的.md文件  
- 简洁高效，可读直观，学习成本低  
- 专注自己的文字内容而不是排版样式，安心写作  
## 用途  
1. IT人士：写日志，技术文稿，记录代码段，撰写文档  
2. 科研人员/学生：撰写科技论文，记笔记  
3. 求职者：制作简历  
4. *其他*...
